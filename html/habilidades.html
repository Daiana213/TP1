HABILIDADES

Ingeniero en Sistemas

Soy graduado de la Universidad Tecnologica Nacional (San Francisco, Cordoba), lo que me hace un profesional integral, destacando por mi capacidad analítica para resolver problemas complejos, mi habilidad para gestionar proyectos tecnológicos y liderar equipos, así como mi aptitud para innovar y comunicar conocimientos de manera efectiva.

Programador

Mi experiencia en programación abarca un amplio espectro de lenguajes, incluyendo Python, Java, Kotlin y JavaScript, lo que me permite adaptarme a diversos proyectos y necesidades. Además, tengo un sólido conocimiento en frameworks populares como Spring Boot, Django y Node.js, facilitando el desarrollo de aplicaciones robustas y escalables. El control de versiones es fundamental en mi flujo de trabajo, utilizando Git y GitHub para la colaboración y el seguimiento de cambios.

Analista de Datos

En el campo del análisis de datos, manejo con destreza herramientas como SQL, Pandas y NumPy para la manipulación y el análisis de información. La visualización de datos es otra de mis fortalezas, utilizando Tableau o Power BI para presentar hallazgos de manera clara y concisa. Además, tengo experiencia en el modelado de datos y el machine learning, aplicando algoritmos de scikit-learn para la construcción de modelos predictivos

Diseño Web

Mi enfoque en el diseño web se centra en la creación de interfaces funcionales y atractivas, utilizando HTML, CSS y frameworks como React o Vue.js. Para el diseño de prototipos y la colaboración, utilizo herramientas como Figma y Adobe XD. Mi formación se complementa con el curso "The Web Developer Bootcamp" en Udemy, que me ha proporcionado una base sólida en el desarrollo web front-end.

Metodologías Ágiles

La gestión de proyectos es otra de mis áreas de especialización, aplicando metodologías ágiles como Scrum y Kanban para la organización y el seguimiento de tareas. Utilizo herramientas de gestión de proyectos como Jira y Trello para facilitar la colaboración y la comunicación dentro de los equipos.
